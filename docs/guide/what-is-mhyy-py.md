# 关于 mhyy.py {#about-mhyy-py}

mhyy.py 是一个基于 [httpx](https://github.com/encode/httpx/) 编写的一系列关于米哈云游的 API 封装。
实现了签到、获取信息等一系列功能。你可以使用 mhyy.py 编写出属于你的自动化程序。

::: tip 只想试一试 ?
跳到 [快速开始](./quick-start.md)。
:::
